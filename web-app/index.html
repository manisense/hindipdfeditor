How it works

Open, edit, export

1

Pick a PDF

Use Android's document picker. The app does not need broad storage permission.

2

Tap or select

Add text by tapping, or use replace mode to cover old text first.

3

Type Hindi

Use your normal Hindi keyboard and preview the edit directly on the page.

4

Export a copy

The original PDF remains unchanged. The edited output is a new file.
