Data Safety Summary

This page summarizes the app's expected Google Play Data Safety declaration in plain English.

Overview

  • Core PDF editing works without developer-hosted document storage.
  • No account is required.
  • No ads SDK is included.
  • No analytics SDK is included in the Android app.
  • The website uses Google Analytics 4 (G-1K5ZEEBHE5) for aggregate page traffic only — not for PDFs you open in the tools.
  • Optional AI enhancement sends the current page image to Google Gemini only by user action.
  • Web Translate runs Opus-MT in your browser; PDF content is not uploaded. A translation model may be downloaded once from Hugging Face.

Data Types

Data Collected or shared? Purpose Required?
PDF page image Shared with Google only if Enhance with AI is used. Text detection for difficult scans. No.
Gemini API key Stored locally on device; sent to Google as part of the API request. Authenticate optional AI enhancement. No.
Opened PDFs and edits Processed locally by the app. Open, preview, edit, and export PDFs. Yes for app functionality.
Opus-MT model weights (web Translate) Downloaded once from Hugging Face to your browser cache; not your PDF. Free offline Hindi→English translation in the browser. No — only if you use Translate on the website.

Security Practices

  • Network requests for optional AI use HTTPS.
  • Gemini API key storage uses Android encrypted storage through Expo SecureStore.
  • The original PDF is not overwritten; exports create a new file.

Data Not Collected By Developer

The developer does not collect name, email, phone number, contacts, location, financial information, advertising identifiers, or analytics events through developer-operated servers or SDKs.

Deletion

There is no developer-hosted account data to delete. Users can delete local app data by clearing app storage or uninstalling the app. Exported PDFs remain wherever the user saved or shared them.
