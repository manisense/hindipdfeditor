Support

Need help with opening a PDF, Hindi text editing, export, optional AI enhancement, or privacy questions? Use the contact options below.

Email

Send support requests to support@hindipdfeditor.com.

Include your Android version, app version, what you were trying to do, and whether the issue happened while opening, editing, OCR detection, or exporting.

GitHub Issues

Technical users can report bugs at github.com/manisense/hindipdfeditor/issues .

Please do not upload private PDFs or identity documents to public issue reports.

Before Sending A File

Many PDFs contain personal, legal, school, financial, or government information. If you need to share a sample, remove private information first or create a dummy PDF that reproduces the issue.

Common Questions

Does the app upload my PDF?

Core editing does not upload documents to developer servers. Optional AI enhancement sends the current page image to Google Gemini only when you explicitly choose that feature.

Does the app overwrite my original file?

No. Export creates a new PDF file.

Why is some old Hindi text blocked from editing?

Some old PDFs use legacy non-Unicode Hindi fonts such as Kruti Dev style encodings. When detected, the app warns you because editing those pages directly can produce incorrect text.

Response Time

We aim to review support messages within a reasonable time, but response times may vary during launch and Play Store review periods.

Cloudflare Email Routing should be configured so support@hindipdfeditor.com forwards to the developer inbox before this page is used in the Play Console listing.
